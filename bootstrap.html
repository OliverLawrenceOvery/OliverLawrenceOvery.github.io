Self-Assessment Revision Aid

Self-Assessment revision aid created using C# and Entity Framework, with a WPF front-end and completed using the SCRUM approach to project management.

Go to GitHub Repository »

Description

This project involved the creation of a revision aid service, allowing the user to store revision notes as well as create their own customisable tests comprised of multiple choice questions for self-assessment purposes. The results from taking these tests would be tracked by the user's personal profile, allowing them to track their progress across the tests.

View details »

Features

The application included numerous features:

  • Ability to create/update/delete personal revision notes.
  • Ability to create/update/delete tests, each of which was comprised of multiple choice questions.
  • Login/register functionality, to allow personalised tracking of user interactions with the application.
  • The ability to graphically represent test results for easy visualisation.

Future Improvements

There are numerous advancements and improvements which could be made to further enhance the functionality of the application:

  • Create a separation of privaleges between a User and Administrator, such that an Administrator can approve submitted tests for shared usage.
  • Develop a means by which revision notes are recommended to the user according to the results of tests.
  • Create more aesthetic layouts for all page views.
